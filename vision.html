Principles

How I build.

01

Field-first, not feature-first.

If a foreman can't use it with one thumb at 6 AM, it's not done. Every design starts at the tailgate meeting.

02

Ship weekly, iterate on the jobsite.

Nothing gets to sit in a branch for a month. Real work runs on the main branch; bugs surface when there's concrete involved.

03

Own the full stack.

Model → agent → API → UI → deployment. Fewer handoffs means faster field feedback loops.

04

Design for the crane operator.

Big type, high contrast, voice-first where it fits. The beautiful polish is a byproduct of making it work with work gloves on.

05

Dual-use everything.

Each internal tool is designed to eventually become a product. Every dashboard is a demo. Every workflow is a sales deck.

06

Keep the humans in the loop.

Agents propose, humans approve. Construction errors cost millions — AI augments the PM, never replaces the judgment.
